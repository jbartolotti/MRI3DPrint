3DPrinting_Brains.docx contains a walkthrough for creating and printing brain models from MRI scans.
make3dBrain is a bash script that is used to convert an anatomical MRI to a 3d model for printing, using either DICOMs, NIFTI, or freesurfer output. Use make3dBrain --help for additional usage info.
BrainBox_Display contains additional 3d-printables for creating a display box to hold the brain model with an integrated LED lamp.
